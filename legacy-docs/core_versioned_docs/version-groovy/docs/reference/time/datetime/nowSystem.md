`nowSystem` provides the current date-time as an Instant, with nanosecond resolution according to the system clock.

## Syntax

```
nowSystem()
```

## Parameters

This method takes no arguments.

## Returns

The current date-time.

## Examples

```groovy order=null
println nowSystem()
```
